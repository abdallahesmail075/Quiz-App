# 🧠 Simple Quiz App

A simple and interactive Quiz Application built entirely from scratch to practice and strengthen my core development skills, logic building, and UI interaction.

---

## 🛠️ Tech Stack
This project was independently developed using:
* **HTML5** & **CSS3** (Structure and Styling)
* **JavaScript** (Game logic, state management, and DOM manipulation)

## 🎯 What I Learned & Practiced
* Managing application state (keeping track of the current question index and score).
* Handling user events and interactive button behaviors.
* Manipulating the DOM efficiently based on user input.
